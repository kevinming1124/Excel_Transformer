# Excel Transformer v0.1 LTS
## 使用說明
1. 打開 exe 資料夾
2. 執行 Excel_Transformer.exe
3. 點擊 "開啟檔案" ，選擇想要轉換的 .xlsx (Excel檔案)
4. 在右側勾選想要保留的欄位
5. 點擊 "輸出檔案" ，等待轉換後，選擇想要儲存的位置以及檔案名稱
6. 完成
## 格式注意
- 因為為測試版本，目前僅開放格式為以下方式呈現的表格
    1. 前三行為該資料的 報表名稱、查詢條件、產生日期
    2. 尚未處理前的資料內容是以 ";" 分隔
    3. 輸入的檔案必須是 .xlsx (Excel檔案)
